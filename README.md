# vlang-noobstuff
playing with vlang

Just a random script to check if two users can "register" based on their ages. No practical use besides learning vlang.

``v run agetest.v
[?] User Name: noob
[?] User age: 1
[?] User2 Name: bob
[?] User2 age: 20
[+]----------------------------
[+] noob is 1 years old.       
[+] noob can register: false   
[+]----------------------------
[+] bob is 20 yeas old.        
[+] bob can register: true     
[+]----------------------------
``
